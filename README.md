# crossword-puzzle

A web-based crossword puzzle application built with HTML, CSS, and JavaScript.

## Features

- Interactive crossword puzzle interface
- User-friendly grid-based layout
- Clues for across and down entries
- Real-time input validation
- Responsive design

## Project Structure

```
crossword-puzzle/
├── Crossword.html    # Main application file
└── README.md         # Project documentation
```

## How to Use

1. Open `Crossword.html` in your web browser
2. Click on a cell to select it
3. Type letters to fill in the puzzle
4. Follow the clues provided for each entry
5. Complete the crossword puzzle

## Technologies Used

- **HTML5**: Structure and markup
- **CSS3**: Styling and layout
- **JavaScript**: Interactivity and logic

## Installation

No installation required. Simply clone the repository and open the HTML file in any modern web browser.

```bash
git clone https://github.com/ritvikvr/crossword-puzzle.git
cd crossword-puzzle
# Open Crossword.html in your browser
```
